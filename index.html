Created by Roman Koshchei for comfortable common punctuation usage. The layout has very low sfb level, only 0.69%. Sfb is when you type next character with the same finger. So low sfb lever improves typing feel and reduces finger effort.

Compare to other layouts Lumin scores better than most of them. Watch stats section. I used genkey analyzer for getting stats and compare with other layouts. Lumin take first place among layouts. As you can see It's much better than QWERTY. So no more reason to stay on QWERTY. Achievement was scoring better than Semimak.

Installation

Windows

  1. Download the following installation bundle: lumin-windows.zip
  2. Extract the zip file somewhere.
  3. If you want use caps lock as backspace open lumod folder, otherwise lumin folder.
  4. Run setup.exe file and install it.
  5. If layout didn't add by itself to current languages, go to language settings and add keyboard to English (United States) language.
